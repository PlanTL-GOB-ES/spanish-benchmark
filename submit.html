Download sample submission files

See the sample files for a hint on naming them.

Make sure that each prediction file is named according to the following:

  • CoNLL-NERC: conll_predictions.txt
  • UD_POS: udpos_predictions.txt
  • MLDoc: mldocs_predictions.txt
  • PAWS-X: pawsx_predictions.txt
  • STS: sts_predictions.txt
  • SQAC: sqac_predictions.json
  • XNLI: xnli_predictions.txt
  • Massive: massive_predictions.txt
