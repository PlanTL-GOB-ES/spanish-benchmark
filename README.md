# EvalES
This website is hosted on the [gh-pages branch](https://github.com/TeMU-BSC/spanish-benchmark/tree/gh-pages).

## Build website
```sh
gulp
```

## Run server
To install dependencies:
```sh
yarn install
```

To run server:
```sh
yarn statrt
```
